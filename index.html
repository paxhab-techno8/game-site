Welcome to Trains Are Cool

Hundreds of games hosted right here. A few tips:

  • Cloak button opens this site inside an about:blank tab so the title and icon look like a blank page.
  • Panic key: press \ (backslash) anytime to instantly redirect to Google.
  • Esc returns to the hub from a game.
  • Search filters by title or description (press / to focus).
  • If a game appears blank, use Open in new tab.

Got an idea or want to suggest a game? Leave a comment on any game page.
